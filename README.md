## Getting Started

### Prerequisites

- Node.js (v18 or later)
- npm (v7 or later)

### Installation

1. Clone the repository:
```shell
$ git clone <repository-url>
$ cd csv-editor
```

2. Install dependencies:
```shell
$ npm install
```

3. Start the development server:
```shell
$ npm run dev
```

The application will be available at `http://localhost:5173`
